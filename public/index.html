HMO API Documentation

Users Routs

Method URL Body Description
GET domain/users/usersList - Get a list of all the users.
GET domain/users/userInfo/{ID number} - Get all information of a specific user by user's ID number.
GET domain/users/coronaInfo/{ID number} - Get corona information of a specific user by user's ID number.
GET domain/users/personalInfo/{ID number} - Get personal information of a specific user by user's ID number.
GET domain/users/usersNotVaccinated - Get the number of users who are not vaccinated.
GET domain/users/sickUsers - Get the number of patients who have previously been sick with Corona.
GET domain/users/usersList?lastName={last name} - Search users by last name. You can also search using a partial last name.
GET domain/users/usersList?mobilePhone={phone number} - Search user by his mobile phone number.
GET domain/users/usersList?city={city} - Search users by city
